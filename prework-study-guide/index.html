HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for Hypertext Markup Language
  • Child elements are nested within parent elements
  • Heading tags should be used in order to improve readability with assitive devices

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The CSSbox model can be used to understand the order of attributes for a css element i.e. Margin > Border > Padding

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git config is used to configure user information used across all local repositories

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are blocks of code that perform a specific task
  • Functions should have a unique name followed by open and closed parenthesis such as "uniqueFunction()"
